# Checklist de reinstalação do projeto Chronos

1. Instalar Node.js (versão 18 ou superior)
2. Instalar Yarn (opcional)
3. Clonar o repositório:
   git clone <URL_DO_SEU_REPOSITORIO>
   cd <nome_da_pasta_do_projeto>
4. Instalar dependências:
   npm install
   # ou
   yarn install
5. Restaurar o arquivo .env com as variáveis do banco e configurações
6. Gerar o client do Prisma:
   npx prisma generate
7. Aplicar as migrações do banco:
   npx prisma migrate deploy
8. Iniciar o projeto:
   npm run dev
   # ou
   yarn dev
