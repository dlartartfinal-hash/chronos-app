# **App Name**: Gestão Chronos

## Core Features:

- Centralized Management Dashboard: Owners gain a 360° view of their business with analytical reports covering sales, stock, customers, and finances.
- Role-Based Access Control: Differentiates user access with Proprietors having full management control and Sellers having restricted PDV access.
- Progressive Web App (PWA): Provides a web application experience with enhanced performance, offline capabilities, and installability, deployable on VPS Hostinger.
- Secure Data Handling: Implements JWT with Passport.js for secure session management and social login, with bcrypt for password hashing.
- Image Storage: Enables image uploads to the VPS server. Serves the images statically via Nginx. Saves public URL to product in database.
- Database Management: Multi-tenant data model on a PostgreSQL database managed with Prisma, ensuring data isolation and security.

## Style Guidelines:

- Primary color: Deep blue (#30475E) for a professional, reliable feel, reminiscent of traditional business applications.
- Background color: Light gray (#F2F4F7), providing a clean and modern backdrop that ensures readability and focuses user attention on the content.
- Accent color: Warm orange (#F05454), strategically used for key interactive elements to draw the user's eye and signify importance.
- Body and headline font: 'Inter' sans-serif font, offering a neutral, modern, and highly readable appearance suitable for both headers and body text.
- Crisp, minimalist icons in a style that balances clarity and visual appeal, enhancing usability.
- A structured, data-dense layout with clear hierarchy, designed to facilitate quick scanning and efficient task completion.
- Subtle transitions and loading animations to enhance user experience without being intrusive.
